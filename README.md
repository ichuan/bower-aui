Bower-aui
=========

Bower repo for the AUI (https://docs.atlassian.com/aui/getting-started.html). Current version: `7.5.1`


Versioning
===========

Since AUI not obey the [semver](http://semver.org/) versioning, here is AUI version to bower version map:

* `5.0.1`    => `5.0.100`
* `5.1`      => `5.1.0`
* `5.1-m1`   => `5.1.1`
* `5.1.0.1`  => `5.1.10`
* `5.2.1-m1` => `5.2.101`
